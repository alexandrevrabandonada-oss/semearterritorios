# Validação da Lista Oficial de Territórios

Este documento orienta a APS e a equipe territorial a validar a lista oficial antes da primeira ação real.

- Nome oficial do bairro/território:
- Cidade:

- Nome alternativo/apelido, se houver:
- Observações:
- Status:
  - [ ] oficial
  - [ ] provisório
  - [ ] revisar
  - [ ] não usar

## Regras de preenchimento

- Não usar nome de rua, casa, família ou endereço como território.
- Não criar território sensível.
- Não misturar bairro oficial com ponto de referência.
- Territórios devem servir para agregação territorial, não para localização individual.
- Se houver dúvida entre nome oficial e apelido, registrar o oficial em `name` e o apelido em `aliases`.

## Perguntas de validação

- Este nome é reconhecido institucionalmente pela APS/equipe territorial?
- Este nome pode ser usado em relatório agregado sem expor pessoa ou família?
- Este território é estável o suficiente para ações, escutas e relatórios?
- Há necessidade de revisar limite, grafia ou agrupamento?
