## Recomendações para Alta Qualidade de Extração
Para que o sistema importe seu relatório sem erros, siga estas diretrizes:

1. **Use exatamente estes títulos:** O sistema busca palavras-chave. Não mude "Atividades realizadas" para "O que fiz na semana".
2. **Evite Tabelas Complexas:** O extrator de texto pode misturar o conteúdo de colunas diferentes. Prefira listas com marcadores (`-` ou `*`).
3. **Não use Imagens com Texto:** O sistema não faz OCR (leitura de imagem). Se precisar registrar algo, digite o texto.
4. **Evite PDFs Escaneados:** Prefira "Salvar como PDF" ou enviar o arquivo DOCX original.
5. **Separação Clara:** Deixe uma linha em branco entre o título da seção e o conteúdo.

**Lembrete de Privacidade:**
Ao escrever o relatório, não inclua CPFs, números de telefone pessoais, e-mails de moradores ou endereços exatos (número da casa). Refira-se aos territórios pelos nomes das comunidades ou bairros oficiais.
