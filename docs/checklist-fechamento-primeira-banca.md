# Checklist de Fechamento — Primeira Banca

Use este checklist antes de considerar a primeira Banca de Escuta encerrada.

## Sistema

- [ ] Ação criada.
- [ ] Fichas numeradas.
- [ ] Fichas digitadas.
- [ ] Escutas revisadas.
- [ ] Possíveis dados sensíveis tratados.
- [ ] Devolutiva gerada.
- [ ] Devolutiva aprovada.
- [ ] Dossiê fechado.
- [ ] Relatório mensal conferido.

## Arquivo físico e memória

- [ ] Arquivos físicos arquivados.
- [ ] Evidências organizadas.
- [ ] Decisão sobre devolutiva pública registrada.
- [ ] Relatório pós-banca preenchido.

## Conferência de privacidade

- [ ] Nenhum CPF registrado.
- [ ] Nenhum telefone real registrado.
- [ ] Nenhum endereço pessoal registrado.
- [ ] Nenhum e-mail pessoal registrado.
- [ ] Nenhum nome completo de pessoa escutada publicado.
- [ ] Nenhuma fala original completa incluída em material público.

## Decisão final

- [ ] Fechamento confirmado.
